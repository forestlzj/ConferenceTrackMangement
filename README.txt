The package includes following:
1. src - The java source and unit-test code
2. pom.xml - Maven config file
3. Talks.txt - A sample input text file

Install
1. Put all above in the same folder and run command "mvn clean install"
2. Run command "java -jar target/TrackManagement-1.0-SNAPSHOT.jar"
3. In the prompt line, input the file path of the talks file (e.g. Talks.txt)
